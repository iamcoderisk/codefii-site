    Codefii Documentation¶

    This is the documentation for Codefii , a Web Framework for Swift that works on macOS and Ubuntu, and all of the packages that Codefii offers.

    Codefii is the most used web framework for Swift. It provides a beautifully expressive and easy to use foundation for your next website or API.

    Getting Started¶

    If this is your first time using Codefii , head to the Install → macOS section to install Swift and Codefii .

    Once you have Codefii installed, check out Getting Started → Hello, world to create your first Codefii app!

    Like Codefii ?¶

    Our small team works hard to make Codefii awesome (and free). Support the framework by starring Codefii on GitHub or donating $1 monthly—it helps us a lot. Thanks!
